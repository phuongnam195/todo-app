# Todo App

## Features
- [x] Completed, Incompleted and Both tasks
- [x] Due date
- [ ] Notification
- [x] Repeat task
- [x] Multi language

## How to build
1. Run command `flutter pub get`
2. Run command `flutter packages pub run build_runner build`
3. For Android, run `flutter build apk`
